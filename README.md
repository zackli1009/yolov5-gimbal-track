##第 1 部分 研究目标

#1. 总体目标
设计并实现一套基于智能车的激光对抗系统，使智能车能够在竞赛场地环境中，准
确识别对手并进行有效 “攻击”，同时具备躲避敌方激光攻击的能力。
1.1.功能目标
1.实现智能车对激光信号的快速、准确识别有效目标，探测距离不低于 5 米，探
测角度覆盖范围达到 180°。
2.智能车可快速规划躲避路径，在 2 秒内完成转向、加速或减速等躲避动作，躲
避成功率不低于 80%。
1.2 性能目标
1.智能车的行驶速度在直线行驶时不低于 1.5m/s，转弯时速度不低于 1m/s，确保
在对抗过程中具有良好的机动性。
2.整个激光对抗系统的功耗控制在 30W 以内，以保证智能车能够持续稳定运行 20
分钟以上 。

##第 2 部分 研究方案

#1.目标跟踪打击系统方案
通过yolo模型用12mm长焦镜头配合PWM数字云台进行目标识别打击。
1.1目标处理算法
使用yolov5视觉算法，通过1000张不同位置目标图片进训练，可以实现在6米以内
的纯色背景下的精准目标识别。其中使用PAN（Path Aggregation Network）网络
结构提高对不同距离，不同角度的目标识别成功率。在代码实际运行中通过对置信
度和IOU值的精调实现更加精准的目标识别。
1.2 目标跟踪方案
基于PWM数字舵机用PID算法控制，添加滤波算法增加平滑度，减小机械抖动的影响
利用opencv的图像处理回传目标中心点，构建二维平面坐标，调整摄像头角度实现
目标跟踪。
运用交互式多模型（IMM）并行多个运动模型，动态加权融合，并利用卡尔曼滤波的预测
能力，在丢失目标后主动跟踪。
1.3 激光发射系统方案
选用半导体激光器作为发射源，搭配高精度的激光准直透镜，确保激光束具有良好
的方向性。通过PWM电机驱动激光发射装置的转动，实现激光发射方向的快速、精
确调整。

#2．躲避路径规划化方案：
利用stm32根据MPU6050实时控制小车的方向，同时利用四路红外传感器实时避开雷
区和墙壁。
#3、基于Python与STM32的串口通信方案：
在上位机jetson中使用python，调用struct库，使用数据包的方式将信息传输给
stm32，在传输过程中确保信息的准确性，防止传输端和接收端的格式不同，而导
致信息乱码，同时stm32使用uart中断，提高stm32的运行效率。

##第 3 部分 技术路线及可行性分析

1.硬件方面：目前市场上已有成熟的光学传感器、半导体激光器和稳定的底控芯片
等元器件，其性能指标能够满足本项目的设计要求。
2.软件方面：yolov5模型目标检测算法，路径规划算法和信号处理算法已有广泛应
用，具有成熟的理论基础和实践经验，通过适当改进和优化，可以应用于本项目的
智能车激光对抗系统 。
3.团队方面：参赛团队成员具备扎实的电子电路、自动控制、图像处理、深度学习
和编程等专业知识，能够完成硬件设计、软件开发以及系统调试等工作 。

##第 4 部分 拟解决的关键问题

#1. 运动目标的准确识别跟踪
实际竞赛环境中，存在多种干扰因素，如环境光、其他参赛车辆的激光信号等。拟
通过优化光电传感器的布局、改进信号处理算法，提高在对目标的识别准确率 。
1.1运动云台控制
智能车在躲避敌方激光攻击和对敌方进行激光打击时，需要快速做出云台响应。关
键在于建立准确的动力学模型，结合实时的路径规划结果，设计高效的电机控制算
法，实现车的快速转向和速度调整，确保躲避动作和打击动作的精准性 。
1.2 激光发射系统的高精度瞄准
激光发射系统需要快速、准确地瞄准目标。拟采用高精度的PWM电机和先进的控制
算法，对激光发射装置的转动进行精确控制。

#2. 智能车快速、精准的躲避控制
根据MPU6050实现小车精准90度左转弯，使小车精准避开雷区，等小车走到对方区
域后，开启红外避障，利用定时器和红外传感实时检测到雷区周围的黑线进行雷区
避障，同时使小车按我们预定的路线进行运动，躲避敌方照射。
